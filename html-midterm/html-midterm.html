Errors:

  1. All HTML documents must start with a !DOCTYPE declaration. I added !DOCTYPE html in line 1.
  2. html language="en" is incorrect, since "language" is not a valid attribute. I fixed this by changing it to "lang".
  3. Moved the HTML lang attribute out of the header.
  4. The head tag was missing its closing tag, I added the closing tag to fix this error.
  5. The h1 tag for "Welcome to My Error-Filled Page" was not closed, so I closed the tag.
  6. The paragraph containing "This is a paragraph with mismatched tags and missing closing tags.", was missing the opening tag for the paragraph and I fixed this error by adding the opening tag.
  7. The list is opened by an ol tag, but closes with a ul tag. Given that the list already has numbers it wouldn't make sense to number the contents twice, so I replaced the ol opening tag with ul to make bullet points.
  8. The paragraph containing "This is another paragraph without proper indentation." is not opened using the p tag, I fixed this error by adding the p tag.
  9. For image.jpeg, they did not provide any alt text in the event the image fails to load. I fixed this by adding in the alt attribute.
  10. The hyperlink closes with img instead of closing with the correct tag. I fixed this by closing with the 'a' element
  11. In the table, where John's age is displayed, the td tag is missing the backslash that would make it close. This was corrected by adding in the backslash and closing the td element.
  12. Table is missing its closing tag, fixed this by adding a closing tag at the end of the table.
  13. The second image uses source when it should be src. I fixed this by replacing source with src.
  14. The text was centered incorrectly by using attributes that are invalid in HTML 5. I fixed this by using opening and closing p tags, and by utilizing css to center the text, style = "text-align: center"
  15. 'bgcolor' is not valid in HTML 5. I fixed this using style="background-color: lightgray;"
  16. The comment was unclosed, I added the proper closing tags to the end.
  17. The link that takes you to the top of the page is missing a closing tag, I fixed this by adding the a closing tag.
  18. The text intended to be underlined has em tags, but they are not properly nested and closed. I fixed this by moving the em closing tag before the p closing tag.
  19. The text meant to be underlined is only emphasized, so I added the u attribute to underline it.
  20. Since there is a description list, using li is not valid. I fixed this by using dt for the title, and dd for the item text.
